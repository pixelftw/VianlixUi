## Usage

To use the `TextReveal` component, follow these steps:

1. **Installation**: Install the necessary package using npm, yarn or pnpm.
   ```bash
   npm install framer-motion
   ```
2. **Tailwind CSS**: Install [tailwind css](https://tailwindcss.com/docs/guides/nextjs) or you can customise the styling according to your ui library

3. **Implementation**: Copy paste the below code in your reposatory

   ```jsx
   function App() {
     const data = useSession();
     return (
       <div>
         <pre>{JSON.stringify(data, null, 2)}</pre>
       </div>
     );
   }

   export default App;
   ```
